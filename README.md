# ctf_app

Create Virtual Environment
###########################

```
sudo apt install python3-venv -y
mkdir ctf_env
python3 -m venv ctf_env
source ctf_env/ctf_env/bin/activate
```
Install CTF APP
###########################
```
git clone https://github.com/Mr-AnyThink/ctf_app.git
cd ctf_app
pip3 install -r requirement.txt
python3 app.py
```
